# 核心来源优先协议

## 1. 固定核心书目

马子密, 傅延龄, 主编. 历代本草药性汇解. 北京: 中国医药科技出版社; 2002.

预期本地文件名可为：

- `历代本草药性汇解 (马子密 傅延龄主编).pdf`
- `历代本草药性汇解.pdf`

该书仍处于版权保护期。公开skill仓库不附整本扫描件；只读取用户依法取得并在当前任务中提供的本地副本。

## 2. PDF来源门禁

开始任何药物考证前：

1. 用 `rg --files -g '*.pdf'` 在工作区和附件目录中检索PDF，再以“历代本草药性汇解”“马子密”“傅延龄”筛选。
2. 检查文字层、总页数、目录、书内页码及扫描清晰度。
3. OCR目录，查找目标药正名及异名；根据下一条目确定目标章节终页。
4. 建立PDF页码与书内印刷页码的映射，并用至少两个页码点复核偏移。
5. 对目标章节逐页OCR，同时查看原图校正异体字、脱字、错行和标点。
6. 输出“已核原图”“仅据OCR”“据汇编转引”“待原书复核”四种校证状态，禁止含混处理。

若本地PDF不存在，只能输出待检索清单和网页线索。必须请用户补充合法副本，不能把在线镜像伪装为已完成纸本阅读。

## 3. 岐黄之术强制检索

站点入口：`https://qihuangzhishu.com`

检索顺序：

1. 搜索目标药正名。
2. 搜索历代异名、俗名和药典习称。
3. 分别组合《神农本草经》《名医别录》《本草经集注》《新修本草》《本草纲目》《本经逢原》《本草求真》等书名。
4. 从药物聚合页逐条打开相关书目页面，记录页面题名、所标古籍、URL、访问日期和可疑异文。
5. 将新增条目放入“待回核队列”，再回查原书影印或可靠校勘本。

已验证的龙胆示例入口：`https://qihuangzhishu.com/763/75.htm`。该页适合发现《名医别录》《本草纲目》《本草经集注》等关联条目，不提供纸本版本和印刷页码证明。

在线镜像可用于确定书中是否收录目标药、发现所汇古籍及建立检索词，但不得据此填写纸本页码。网页内容与本地PDF不一致时，保留差异并以可靠版本校证，不静默合并。

## 5. 最小来源记录

每条发现至少记录：

| 字段 | 要求 |
|---|---|
| 目标药名 | 正名及检索所用异名 |
| 来源类型 | 原书、校勘本、汇编本、岐黄之术或网页镜像 |
| 书名与版本 | 能确认多少写多少，不推测 |
| 定位 | 书内页码、PDF页码或网页URL，三者不得混写 |
| 条文 | 短引文或忠实释义 |
| 证据用途 | 名称、基原、药用部位、性味、归经、毒性、功效、炮制或禁忌 |
| 校证状态 | 已核原图、仅据OCR、据汇编转引或待原书复核 |
| 访问日期 | 网页来源必填 |

只有“已核原图”或可靠校勘本支持的内容，才可作为直接引文和页码证据；其余材料用于发现、比较和提出待证问题。
