# 可配置来源优先协议

## 1. 建立来源矩阵

每个目标药物都重新配置来源，不假定任何单一汇编本必然收录。至少列出：

| 类别 | 变量 | 作用 |
|---|---|---|
| 原始与校勘来源 | `LOCAL_PRIMARY` | 原书影印、可靠校勘本、用户合法提供的扫描件；支持原文和页码 |
| 核心汇编来源 | `LOCAL_COMPENDIA` | 正式出版的汇编本或考证专著；用于建立书目和异文线索 |
| 法定与官方来源 | `OFFICIAL_STANDARDS` | 药典、部颁/地方标准、监管文件；支持法定基原与标准事实 |
| 发现来源 | `DISCOVERY_SOURCES` | 岐黄之术、图书馆目录、在线镜像；只用于发现条目 |
| 现代研究来源 | `MODERN_DATABASES` | 中文数据库、PubMed等；支持现代学术判断和证据评价 |

《历代本草药性汇解》可在用户合法提供且与目标药物相关时配置为 `LOCAL_COMPENDIA`，但不是所有任务的固定前提。书目信息：马子密, 傅延龄, 主编. 历代本草药性汇解. 北京: 中国医药科技出版社; 2002。

仍受版权保护的整本扫描件不得放入公开skill仓库。只在用户依法取得并提供的当前任务副本上校读。

## 2. 本地PDF门禁

对每一本配置来源分别执行：

1. 用 `rg --files -g '*.pdf'` 清点工作区和附件目录，按书名、作者、目标药正名及异名匹配。
2. 检查文字层、总页数、目录、书内页码、缺页和扫描清晰度。
3. 在目录、索引和全文中分别检索 `TARGET_HERB` 与 `TARGET_SYNONYMS`，保存实际检索词。
4. 未命中时记录“书名、版本、检索范围、检索词、未命中”，然后转入下一来源；不得把未命中误写成“该书未曾记载”的历史结论。
5. 命中时根据相邻条目确定章节起止，建立PDF页码与书内印刷页码映射，并用至少两个页码点复核偏移。
6. 对目标章节全部连续页OCR并查看原图，校正异体字、脱字、错行、栏序和标点。
7. 将文字标为“已核原图”“可靠校勘本”“仅据OCR”“据汇编转引”“待原书复核”之一。

若关键本地PDF不存在，只能输出缺口清单、候选书目和网页线索。不得把在线镜像描述为已完成纸本阅读，也不得生成纸本页码。

## 3. 岐黄之术通用检索

入口：`https://qihuangzhishu.com`

使用站内检索或搜索引擎依次组合：

```text
site:qihuangzhishu.com "TARGET_HERB"
site:qihuangzhishu.com "TARGET_SYNONYM"
site:qihuangzhishu.com "TARGET_HERB" "BOOK_TITLE"
site:qihuangzhishu.com "TARGET_SYNONYM" "BOOK_TITLE"
```

操作要求：

1. 先搜正名，再逐一搜索异名、俗名、历史习称和可能混用品名。
2. 将命中条目所关联的古籍书名加入候选书目清单，再分别组合检索。
3. 记录页面题名、网页标注书名、URL、访问日期、短引文/释义和可疑异文。
4. 每条新增材料进入“待回核队列”，回查原书影印、可靠校勘本或正式整理本。
5. 页面标签不是纸本版本证明；网页没有书内印刷页码时不得自行补写。

## 4. 其他在线目录与镜像

在线目录和镜像只有在来源矩阵中明确列出后才使用。例如，《历代本草药性汇解》公开目录镜像可从以下索引发现条目：

`https://culture.pkstate.com/tcm3/lidaobencaoyaoxinghuijie/index.html`

该类页面可用于判断可能收录、发现所汇古籍及生成检索词，但不能替代本地合法PDF或可靠版本。网页与纸本、不同整理本不一致时，分别记录差异，不静默合并。

## 5. 证据闭环

每条线索必须沿以下链路闭合：

```text
发现来源 -> 候选书目/条文 -> 原书或可靠版本回核 -> 证据台账 -> 正文/表格 -> AMA书目
```

最小记录字段：

| 字段 | 要求 |
|---|---|
| 待证问题 | 名称、基原、部位、产地、炮制、药性、功效、毒性等 |
| 目标名称 | 正名及本次使用的检索异名 |
| 来源与层级 | 来源类型、A-E层级、书名、作者和版本 |
| 定位 | 书内页码、PDF页码、卷次或URL分别记录 |
| 原文/释义 | 短引文或忠实释义；标明直接引文或转述 |
| 证据用途 | 拟支持的表格字段或正文判断 |
| 校证状态 | 已核原图、可靠校勘本、仅据OCR、据转引、待回核或未命中 |
| 版本差异 | 异文、缺页、栏序或版本不明情况 |
| 访问日期 | 所有网页来源必填 |

只有已核原图、可靠校勘本、正式出版物或官方来源支持的内容，才能作为关键直接引文和确定页码证据。无法闭环的线索应留在“证据缺口”，不得用确定语气写入结论。

## 6. 来源门禁输出

进入正式写作前输出一页来源状态摘要：

- 已命中的核心来源及页码范围。
- 未命中的来源、实际检索词和限制。
- 已回核的网页线索数量与仍待回核条目。
- 版本冲突和拟采用版本。
- 可支持的研究问题与暂不可回答的问题。

该摘要是每种药都可复用的工作起点，而不是论文正文的一部分。
